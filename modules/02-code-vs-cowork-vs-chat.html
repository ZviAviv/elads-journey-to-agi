Three doors. Same Claude. Different jobs.

Anthropic ships three ways to talk to Claude. Picking the wrong one is the most common reason people say "I tried it, it did not work."

The three surfaces

Claude.ai chat. The browser tab. Stateless, friendly, no access to your computer. Best for thinking out loud, learning, drafts, paste-and-go conversations.

Cowork. Anthropic's surface for non-developer knowledge work. It connects to Google Drive, Gmail, DocuSign, FactSet. It manipulates documents, extracts data, runs multi-step workflows over your files. No code required, no terminal, no git.

Claude Code Desktop. The app. Reads your folders, edits files, runs commands, uses git, builds software. The full agent in a window with diffs, terminal, and a preview pane.

The decision in one sentence

If you are thinking, use chat. If you are doing office work, use Cowork. If you are building or fixing something on your computer, use Claude Code.

How they coexist

You will end up with all three open. That is normal and good. Sketch an idea in chat. Pull research into Cowork. Build the actual thing in Code. They each cost the same in your plan, so the only question is fit.

Best practices

Do

  • Default to chat for "explain this" or "what would you do here." Cheapest cognitively.
  • Use Cowork when the work lives in Drive, Gmail, or a spreadsheet you do not want Claude to clone locally.
  • Use Claude Code when the work lives in a folder on your machine and would benefit from version control.
  • Move work between surfaces deliberately. Chat to plan, Code to build, Cowork to ship the report.
  • Bookmark all three. Friction kills the habit.

Don't

  • Paste your codebase into chat. Chat cannot run it, cannot save changes, and forgets after a tab close.
  • Try to build a real product in Cowork. Wrong tool, wrong feedback loop.
  • Open Claude Code to ask "what is a closure." That is a chat question.
  • Assume one surface knows what you did in another. They do not share memory.
Newbie trap Hearing "Claude Code" and assuming it is for developers only. The Desktop app is genuinely usable by anyone who can install software. The "Code" name is about what it can do, not about who you have to be.

Exercise. Pick a real task from your week. Write it in one sentence on a sticky note. Now write next to it which of the three surfaces fits best, and why. Open that one. Do the task. Notice if you wanted to switch halfway. That is information.
