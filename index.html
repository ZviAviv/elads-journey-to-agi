to GAI.

Elad, this site exists for you.

By the end you will have installed Claude Code Desktop, written your first real prompt, picked the right model and mode for the task, and known when to reach for a skill, a plugin, or a subagent.

Read it linearly the first time. Come back to any module when you forget something.

What is GAI

It does not stand for anything. Pretend it stands for whatever you want. General Agentic Imagination. Genuinely Annoying Intelligence. Get Aviv's Insight. Pick one. The point is the journey.

Who this is for

You are comfortable with computers. You have used ChatGPT or Claude in the chat window. You have not written code, or you have written some, but agents are still new. You want to use Claude Code Desktop, the app, and you want to use it well.

This is not a curriculum to admire. Read a module, then go open Claude Code and try the exercise. The site waits.

How the site works

Two ways to read it. Use both:

  • Linear walk: use the Next button at the bottom of each module. Or press → on your keyboard.
  • Jump anywhere: the sidebar on the left lists every module. Click whichever one you want.

The bar across the top tracks how far through the page you have scrolled. The Glossary defines every term Elad does not need to memorize. The Tips Index pulls every Do and Don't into one cheatsheet.
